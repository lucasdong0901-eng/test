# Tetris (Vanilla JS)

브라우저에서 바로 실행되는 **싱글 파일 테트리스**입니다.  
화면 흔들림(스크린 쉐이크), 고스트 블록(착지 예상 위치), 점수/라인/레벨 시스템이 포함되어 있습니다.

---

## Demo
- `index.html`을 브라우저로 열면 바로 플레이 가능합니다.

---

## Features
- ✅ 10x20 테트리스 보드
- ✅ 7가지 테트로미노(I, J, L, O, S, T, Z)
- ✅ 고스트 블록(착지 위치 표시)
- ✅ 하드 드랍 / 소프트 드랍
- ✅ 라인 클리어 + 점수/레벨 시스템
- ✅ 스크린 쉐이크(라인 클리어/하드드랍/게임오버 타격감)
- ✅ 일시정지

---

## Controls
- **← / →** : 이동  
- **↓** : 소프트 드랍(한 칸 내려가기)  
- **↑ 또는 R** : 회전  
- **Space** : 하드 드랍(바닥까지 즉시)  
- **P** : 일시정지/해제  
- **Enter** : 재시작

---

## How to Run
1. 이 레포를 클론하거나 `index.html`을 다운로드합니다.
2. `index.html`을 더블클릭(또는 브라우저로 열기)하면 실행됩니다.
